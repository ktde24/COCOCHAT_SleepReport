# COCOCHAT_SleepReport

### FLY AI 프로젝트
#### 수면 Report 페이지
- 아이의 나이, 이름, 낮잠 시간, 낮잠 횟수, 밤잠 시간 데이터를 가져와서 일주일 동안의 수면 그래프 표시
- 또래의 적정 수면 시간과 비교한 그래프 표시
- 수면 데이터를 챗봇에 쿼리문으로 전달하여 응답을 받아오고, 그 응답을 사용자에게 제공
